# 🎓 AI Placement Intelligence Platform

An AI-powered Placement Intelligence Platform built using **Python, Streamlit, Scikit-learn, SQL, and Power BI**. The application predicts placement opportunities based on student academic performance and skills, recommends suitable companies, analyzes uploaded resumes, identifies skill gaps, and provides placement readiness insights through an interactive dashboard.

---

## 🚀 Features

* 🤖 AI-Based Company Prediction
* 📄 Resume PDF Analysis
* 📊 Placement Probability Prediction
* 🎯 Placement Readiness Score
* 🏆 Student Ranking
* 💼 Top Company Recommendations
* 📚 Skill Gap Analysis
* 📜 Prediction History
* 📥 Downloadable Placement Report
* 📈 Interactive Streamlit Dashboard

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/anshikapurwar63-spec/AI-Placement-Intelligence-Platform.git
```

2. Open the project folder.

3. Install the required libraries:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
streamlit run app_streamlit.py
```

---

## 📊 Project Workflow

1. Enter student academic details.
2. Upload a resume in PDF format.
3. Predict the most suitable company.
4. Analyze resume skills.
5. View placement probability and readiness score.
6. Review company recommendations and skill gap analysis.
7. Download the placement report.
